  @par Demo Description 
This demo shows:  
    - how to run Bk72xx system based on a hello world applicaiton program. 
      Include the following specifical function:
       - how to make a thread for helloworld Application;
       - how to Print logs through BK72xx UART2(system debug serial port of BK72xx);
       - how to flip a LED(connected to BK72xx GPIO) ON or OFF  status by calling BK72xx API;
       - how to Hang up the current thread for a while by calling Bk72xx API.
